\chapter{期权定价方法与算法推导}
\label{chap:analysis}

在明确了3/2波动率模型的随机微分方程与积分方差特性后，本章将详细介绍实现期权定价的具体数值算法与推导步骤 。主要涵盖蒙特卡洛模拟框架（包括条件模拟与精确模拟）以及基于傅里叶变换的频域定价方法。

\section{蒙特卡洛模拟框架}
\label{sec:monte_carlo_framework}

\subsection{蒙特卡洛模拟}


\subsection{条件蒙特卡洛模拟}
条件蒙特卡洛模拟旨在通过生成波动率路径来逼近积分方差，其主要执行步骤如下 ：
\begin{enumerate}
  \item 路径模拟： 在时间网格 $0 \le t_1 \le t_2 \le \dots \le T$ 上，模拟波动率过程 $\sigma_t$ 的离散路径，并获取终端波动率 $\sigma_T$。
  \item 积分计算： 利用数值积分法则（如梯形法则或辛普森法则 Trapezoidal / Simpson’s rule），对模拟出的方差路径进行时间积分，从而获取整个时间段内的积分方差 $V_T$。
\end{enumerate}

\subsection{精确蒙特卡洛模拟}
为了彻底消除时间离散化误差，精确模拟方法直接对终端状态进行抽样，而不依赖于路径的逐步生成。其核心步骤为：
\begin{enumerate}
  \item 终端方差抽样： 由于终端方差 $v_T$ 的转移密度满足非中心卡方分布，我们可以直接从其解析分布中抽取 $v_T$ 的样本。
  \item 条件积分方差抽样： 在已知 $v_T$（或等价地，终端波动率 $\sigma_T$）的条件下，从 $V_T|v_T$ 的数值累积分布函数 (CDF) 中抽取积分方差 $V_T$ 的样本。这一步涉及复杂的内层拉普拉斯逆变换 (Inverse Laplace transform) 操作 。
\end{enumerate}

\subsection{定价计算与聚合}
无论采用条件MC还是精确MC获取了积分方差，后续的期权价格计算步骤是通用的 ：
3. 参数转换： 计算给定 $v_T$ 和 $V_T$ 条件下的标的资产期望值 $E(S_T|v_T, V_T)$，并确定有效波动率 (effective volatility)，通常表示为 $\rho^*\sqrt{V_T/T}$ 。
1. 期权估值：
    *路径一（价格抽样）： 根据对数正态分布或正态分布进行价格抽样，计算终端的期权收益 。路径二（解析公式）： 直接将修正后的参数代入 Bachelier 或 Black-Scholes-Merton (BSM) 期权定价公式中 。具体而言，设修正后的初始价格为 $S_0 := E(S_T|v_T)$，修正波动率为 $\sigma := \rho^*\sqrt{V_T/T}$ 。取均值： 对所有独立模拟路径得出的期权价格取平均值，得到最终的期权期望现值 。
    *路径二（解析公式）： 直接将修正后的参数代入 Bachelier 或 Black-Scholes-Merton (BSM) 期权定价公式中 。具体而言，设修正后的初始价格为 $S_0 := E(S_T|v_T)$，修正波动率为 $\sigma := \rho^*\sqrt{V_T/T}$ 。
1. 取均值： 对所有独立模拟路径得出的期权价格取平均值，得到最终的期权期望现值 。

\section{傅里叶变换方法}
除了时域上的蒙特卡洛模拟，在特征函数已知的前提下，频域方法（傅里叶变换）提供了另一种高效的定价途径。
\begin{enumerate}
  \item 快速傅里叶变换 (FFT)： 通过将期权定价公式转化为傅里叶空间中的积分，FFT能够一次性计算出多个执行价格下的期权价格，极大地提高了计算速度 。然而，在处理极度临近到期 (near expiration) 的期权时，积分的振荡性质会导致定价失效或精度严重下降 。
  \item 傅里叶-余弦变换 (Fourier-Cosine 方法)： 为了克服上述缺陷，本研究重点引入并考察了基于条件傅里叶-余弦变换的定价方法 。该方法通过余弦级数展开来逼近目标概率密度函数。理论与后续数值结果均表明，Fourier-Cosine变换不仅能保持相对较快的计算速度，还能在更为极端的参数条件下实现精确的期权定价 。
\end{enumerate}
